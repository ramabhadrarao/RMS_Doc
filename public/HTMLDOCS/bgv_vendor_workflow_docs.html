BGV Vendor Workflow: Full Documentation

A comprehensive guide for users and developers on the BGV Vendor creation process.

Form Sections & Input Fields

Objective: To capture all necessary details for a new BGV Vendor, from initial selection to contact persons and documentation.

1. Client Selection

  • Select Client (Dropdown, Required)

2. Vendor Details

  • Vendor Name (Text, Required)
  • Website (Text, Optional)

3. BGV Vendor & Address Details

  • Address (Textarea)
  • Billing Address (Textarea)

4. Billing Information

  • GST Number (Text)
  • Invoice Payment Terms (Number, in days)

5. Documents Section

This section allows for uploading multiple documents. Each document entry has the following fields:

  • Document Name (Text)
  • Validity (Checkbox)
  • Start Date (Date, if validity is checked)
  • End Date (Date, if validity is checked)
  • File Upload (File)

6. Vendor SPOC (Single Point of Contact) Details

Allows for adding multiple SPOCs for the vendor.

  • Name (Text)
  • Email (Email)
  • Phone (Phone)
  • Designation (Text)
  • Location (Text)
  • Status (Dropdown: Active/Inactive)
  • Functional Roles (Multi-select)
  • Accounts Handled (Multi-select, Optional)
