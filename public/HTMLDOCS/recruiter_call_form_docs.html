Recruiter Call Form: Full Documentation

A comprehensive, line-by-line guide for the Recruiter Call Form, detailing the complete workflow and data structure.

Line-by-Line Workflow

This table is a direct, step-by-step representation of the workflow defined in the RecruiterCallForm Workflow.pdf document. No steps have been skipped or consolidated.

# Field / Action Details (Input, Options, Required) Behavior & Workflow
Main Form: RecruiterCallForm - Initial Fields
1Candidate NameInput: Text
Required: No		Free text input.
Updates: formData.candidateName
2Contact NumberInput: Text (tel)
Required: No		Free text input.
Updates: formData.contactNumber
3EmailInput: Text (email)
Required: No		Free text input.
Updates: formData.email
Candidate Response Section
4 Are you currently looking for a job change? Input: Radio Buttons
Required: Yes
Options:
- Yes
- No
- No response / Wrong Number
- Disconnected
- Asked to call later		 Triggers conditional sections based on selection.
  • If Yes: Shows RecruiterForm.
  • If No: Shows Reason for "Not Interested".
  • If No response / Disconnected: Shows Close Interaction.
  • If Asked to call later: Shows Schedule Follow-up.
Conditional Flow: If Candidate Response = "No"
5 Reason for "Not Interested" Input: Dropdown
Required: Yes
Options:
- Not actively looking
- Already working and happy
- Only open to permanent roles
- Not interested in current job location
- Salary not matching expectations
- Already got offer / interview pipeline		 The selected reason triggers a specific sub-flow.
  • If "Only open to permanent roles", "Not interested...", or "Salary not matching...": Shows the nested RecruiterForm.
  • If "Already got offer / interview pipeline": Shows the Offer/Interview Pipeline section.
  • If "Not actively looking" or "Already working and happy": Shows the Close Interaction section.
Conditional Flow: If Candidate Response = "Asked to call later"
6Schedule Follow-upInput: Date + Time
Required: Yes		Shows date and time fields. On submission, stores the scheduled date/time and hides other sections.
Conditional Flow: If Candidate Response = "No response / Wrong Number" or "Disconnected"
7Close InteractionInput: Radio Buttons
Required: Yes
Options:
- FTE pipeline
- Passive pipeline
- Immediate match (if any)		Shows options to mark the interaction status. No further form sections are shown.
Trigger for Nested Form: RecruiterForm
8Show RecruiterFormTriggered When:
- Candidate response is "Yes".
- Candidate response is "No" AND reason is "Only open to permanent roles", "Not interested in current job location", or "Salary not matching expectations".
Behavior: Shows the entire nested form below to collect detailed candidate info.
Nested Form Step 1: Candidate's Education
9Select all education levels completedInput: Checkboxes
Options: SSC, Intermediate, Diploma, Graduation, PG		On selection, shows dropdowns for each selected level with options: Both (default), Regular, Correspondence.
10Academic Program TypeInput: Radio Buttons
Options: Regular, Non-Regular		If "Non-Regular", shows a text input for remarks (e.g., "Distance education, Gap reason").
11Highest Qualification & Completion DateInput: Text & Month Picker
Required: Yes		Shows only after selecting an academic program type.
Nested Form Step 2: Marital Status
12Marital StatusInput: Radio Buttons
Required: Yes
Options: Married, Unmarried		Shown after entering highest qualification. Required for relocation planning.

  • If No: Shows Preferred Job Location & Remarks fields.

23Current CTC (Per Month)Input: NumberFree text input.
24Take Home (Per Month)Input: NumberFree text input.
25Expected CTC / Offered Salary (Per Month)Input: NumberFree text input.
26Variable Salary (Per Month)Input: NumberFree text input.
Nested Form Step 6: Work Status & Offer Details
27Work StatusInput: Dropdown
Required: Yes
Options: Working, Not Working, Fresher
  • If Working: Shows Client/Payroll Company, Notice Period, Dates, Status, Resignation Plan.
  • If Not Working: Shows Start/Last Working/Expected Joining Date.
  • If Fresher: Shows Internship questions.
Nested Form Step 7: Resignation Reason (if Working)
28Reason for Job ChangingInput: Dropdown
Options: Bench, No Project, Career Growth, etc.		If "Other" is selected, shows an input to specify a custom reason.
29Do You Have Resignation Proof?Input: Dropdown (Yes/No)If "Yes", allows file upload. If "No", shows a remarks input field.
Nested Form Step 8: Offer/Interview Pipeline (if Working)
30Do You Have Any Offers in Hand?Input: Dropdown (Yes/No)If "Yes", shows an input for details and allows file upload for the offer proof.
31Interviews in PipelineInput: Dropdown (Yes/No)If "Yes", shows an input for interview company names.
32Why Are You Interested in This Job?Input: TextareaFree text input.
Conditional Flow: If Response="No" and Reason="Already got offer / interview pipeline"
33Skill is critical. Try to pitch better role?Input: Radio Buttons (Yes/No)
  • If Yes: Shows Reference Check section.
  • If No: Hides both RecruiterForm and Reference Check.
34Reference Check (if pitching better role)Input: Radio Buttons (Yes/No)If "Yes", shows inputs for Reference Name and Phone Number. If "No", hides both.
Final Actions
35-36Schedule Follow-upInput: Date Picker, Time PickerFree date and time selection.
37Close InteractionInput: Radio Buttons
Options: FTE pipeline, Passive pipeline, Immediate match		Allows recruiter to categorize the lead.
38Submit for Lead ReviewButton: Blue, full-widthSubmits the form. In a real app, this would send data to the backend, log to CRM, update candidate status, and schedule follow-ups.
